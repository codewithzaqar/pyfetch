# Pyfetch

A Python-based command-line system information tool inspired by Neofetch.

## Installation 

```bash
pip install pyfetch
```

## Usage
Run the tool with
```bash
pyfetch
```
### Option
- `--no-art`: Disable ASCII art.
- `--fields`: Specify fields to display (e.g, `OS`,`CPU`,`Memory`).

Example:
```bash
pyfetch --fields OS,CPU,Shell --no-art
```
